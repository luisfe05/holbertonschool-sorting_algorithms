# C - Sorting algorithms & Big O

## Description
This project covers the implementation of sorting algorithms in C
and the analysis of their time complexity using Big O notation.

## Files

| File | Description |
| --- | --- |
| `0-bubble_sort.c` | Function that sorts an array using Bubble sort |
| `0-O` | Big O notations for Bubble sort: O(n) / O(n^2) / O(n^2) |
| `1-insertion_sort_list.c` | Function that sorts a doubly linked list using Insertion sort |
| `1-O` | Big O notations for Insertion sort: O(n) / O(n^2) / O(n^2) |
| `2-selection_sort.c` | Function that sorts an array using Selection sort |
| `2-O` | Big O notations for Selection sort: O(n^2) / O(n^2) / O(n^2) |
| `3-quick_sort.c` | Function that sorts an array using Quick sort (Lomuto partition) |
| `3-O` | Big O notations for Quick sort: O(nlog(n)) / O(nlog(n)) / O(n^2) |
